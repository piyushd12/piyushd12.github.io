## Projects

### Infrastructure Change Detection in Geospatial Imagery

Built a change detection system that compares pre and post-disaster satellite images to identify and classify building damage. Used DINOv2 for feature extraction and VM-UNet for segmentation, fine-tuned on the xBD dataset. The output is color-coded heatmaps showing damage severity across affected areas.

**Tech stack:** Python, PyTorch, DINOv2, VM-UNet, OpenCV  
**GitHub:** [github.com/piyushd12](https://github.com/piyushd12/Infrastructure_Change_Detection_in_Geospatial_Imagery)

---

### Sign Language Translation using Sequence Learning

A pipeline that translates hand gestures to text in real time using MediaPipe for hand tracking and LSTM for sequence learning. Also built a data collection tool so anyone can record and label their own gestures, which generalizes to regional or custom sign languages.

**Tech stack:** Python, TensorFlow, MediaPipe, LSTM, OpenCV  
**GitHub:** [github.com/piyushd12](https://github.com/piyushd12/Sign-Language-Translation)

---

### Timetable Generation using Genetic Algorithm    

A genetic algorithm that generates class and lab schedules under hard constraints (no room conflicts, no teacher overlaps, balanced slots). Fitness functions handle the tricky parts. Configurable for different departments, courses, and time structures.

**Tech stack:** Python, Genetic Algorithms  
**GitHub:** [github.com/piyushd12](https://github.com/piyushd12/genetic-timetable-generator)
