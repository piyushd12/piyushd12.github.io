# Images Directory

Place your site images here:

- `avatar.png` — Profile photo used on the homepage (profileMode) and header label.
  Recommended size: 400×400px, square, transparent or white background.

- `og-image.png` — Default Open Graph image for social sharing (1200×630px).

- `favicon.ico` — Browser tab favicon.
- `favicon-16x16.png` — 16×16 favicon variant.
- `favicon-32x32.png` — 32×32 favicon variant.
- `apple-touch-icon.png` — iOS home screen icon (180×180px).

## covers/

Post cover images live in the `covers/` subdirectory.
See `static/images/covers/README.md` for conventions.
