# Post Cover Images

Place post cover images here. Naming convention:

- Use kebab-case matching the post filename.
  e.g., post slug `sample-post-go` → cover image `go-api.png`
- Recommended size: **1200×630px** (standard Open Graph aspect ratio).
- Supported formats: PNG, JPEG, WebP.

## Referencing in frontmatter

```yaml
cover:
  image: /images/covers/go-api.png
  alt: "Descriptive alt text"
  caption: "Optional caption shown below the image"
```
